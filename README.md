<h1>Security Audit</h1>


<h2>Description</h2>
This Project was a security audit evaluation for an organization's information systems to ensure they adhere to security policies, standards, and requirements. The objectives were as follow:

Identify Vulnerabilities: Helps uncover weaknesses that could be exploited by attackers.

Compliance: Ensures adherence to regulations and standards like GDPR, HIPAA, and PCI DSS.

Risk Management: Assesses risks and helps implement measures to mitigate them.

Improve Security Posture: Provides actionable insights to enhance overall security.

Data Protection: Ensures sensitive data is properly safeguarded.
